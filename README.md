# 📊 BukuKas UMKM - Solusi Pembukuan Digital
Aplikasi Pembukuan UMKM berbasis web yang dirancang untuk membantu pengusaha kecil mengelola keuangan dengan mudah, aman, dan profesional. Aplikasi ini merupakan **Progressive Web App (PWA)**, sehingga dapat diinstal di perangkat Android, iOS, Windows, maupun macOS tanpa melalui App Store.
## ✨ Fitur Utama
 * **Dashboard Visual**: Grafik pendapatan dan pengeluaran secara real-time.
 * **Manajemen Transaksi**: Catat, edit, dan hapus data transaksi dengan cepat.
 * **Kategori Kustom**: Kelola kategori pemasukan dan pengeluaran sesuai kebutuhan bisnis.
 * **Database Lokal**: Menggunakan *IndexedDB* untuk penyimpanan data yang aman di perangkat pengguna (Offline Ready).
 * **Ekspor Laporan**: Unduh laporan keuangan dalam format **PDF** dan **Excel (.xlsx)**.
 * **Desain Responsif**: Tampilan optimal untuk Desktop, Tablet, maupun Smartphone.
## 🚀 Teknologi yang Digunakan
 * **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+).
 * **Icons & Charts**: Lucide Icons, Chart.js.
 * **PWA**: Service Workers & Web App Manifest.
 * **Data Export**: SheetJS (XLSX) & jsPDF.
## 📥 Cara Instalasi (PWA)
 1. Buka URL aplikasi di browser HP atau Komputer Anda.
 2. **Android/Chrome**: Klik ikon titik tiga di pojok kanan atas, lalu pilih **"Instal Aplikasi"**.
 3. **iOS/Safari**: Klik ikon **Share**, lalu pilih **"Add to Home Screen"**.
 4. **Desktop**: Klik ikon instalasi di baris alamat browser (Omnibox).
## 🛠️ Pengembangan
Aplikasi ini berjalan sepenuhnya di sisi klien (*client-side*). Tidak ada data yang dikirim ke server luar, semua data keuangan Anda tersimpan aman di database browser perangkat Anda sendiri.
